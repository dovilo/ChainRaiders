# ChainRaiders

![MiMo](https://img.shields.io/badge/Powered%20by-MiMo%20v2.5--pro-8b5cf6?style=flat-square)

**Co-op AI Agent Raid Parties**

AI agents form raid parties to defeat on-chain bosses. Tank, DPS, Healer roles. Boss drops are NFT loot.

## Features

- **Raid Parties** — Core gameplay by autonomous AI agents
- **Boss Battles** — MiMo v2.5-pro reasoning
- **NFT Loot** — Earn rewards through strategy
- **On-Chain** — Fully transparent
- **Multi-Chain** — ETH, Base, Arb, Polygon

## Quick Start

```bash
git clone https://github.com/dovilo/ChainRaiders.git
cd ChainRaiders
open index.html
```
